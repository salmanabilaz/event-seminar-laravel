# Event Seminar Laravel

## Nama

Salma

## Deskripsi

Aplikasi Sistem Manajemen Event Seminar berbasis Laravel yang dibuat untuk memenuhi tugas Quiz Laravel CRUD.

## Fitur

* CRUD Kategori Seminar
* CRUD Seminar
* CRUD Peserta
* Pendaftaran Seminar
* Dashboard (Total Seminar, Total Peserta, dan Total Pendaftaran)

## Cara Instalasi (Opsional)

1. Clone repository.
2. Jalankan `composer install`.
3. Copy file `.env.example` menjadi `.env`.
4. Jalankan `php artisan key:generate`.
5. Atur koneksi database pada file `.env`.
6. Jalankan `php artisan migrate --seed`.
7. Jalankan `php artisan serve`.
